# Интернет-магазин (E-commerce Core)

## Описание
Ядро интернет-магазина с основными сущностями:
- Товары (Product)
- Категории (Category)

## Функциональность
- Создание товаров и категорий
- Автоматический подсчет количества товаров и категорий

## Установка
1. Создайте виртуальное окружение: `python -m venv venv`
2. Активируйте его: `source venv/Scripts/activate` (Windows Git Bash)
3. Установите зависимости: `pip install pytest flake8`

## Запуск
```bash
# Запуск основной программы
python src/main.py

# Запуск тестов
pytest tests/

# Проверка стиля кода
flake8 src/
